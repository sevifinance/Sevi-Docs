# Payments Overview

From your **Agent Accounts**, tap on **`Payments`** to see a list of all payment requests created for your buyers.  


## 1. View All Payments
- The **Payments screen** shows all payment requests created by you for your customers.  
- Each order will display:  
  - **Buyer Profile**  
  - **Order Value**  
  - **Created Date**    
  - **Seller Name**  
  - **Order Status**  
  - **Order Number**
